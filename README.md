# Pizza-dashboard
Pizza Sales Analysis Dashboard (Power BI)
**Project Overview**

This project focuses on analyzing pizza sales data to uncover key business insights such as sales trends, customer ordering behavior, and product performance. The dashboard is built using Power BI and provides interactive visualizations to support data-driven decision-making.

**Objectives**
Analyze total orders across different days and months
Identify top-performing and low-performing pizzas
Understand sales contribution by pizza category
Track overall sales performance and trends

**Tools & Technologies Used**

SQL Server – Data storage and initial data extraction
Power BI Desktop – Data transformation, modeling, and visualization
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Measures and calculations
CSV File – Raw data source

**Data Workflow**

Imported raw data from CSV file
Loaded data into SQL Server for structured storage
Extracted data from SQL Server into Power BI
Performed data cleaning and transformation using Power Query
Built data model (Star Schema approach)
Created calculated measures using DAX
Designed interactive dashboards

**Dashboard Features**

Page 1: Sales Overview
Total Orders by Order Day
Monthly Trends for Total Orders
Percentage of Sales by Pizza Category
Total Pizzas Sold by Category
Page 2: Product Performance
Best-Selling Pizzas
Worst-Selling Pizzas

 **Key Insights**
 
Identified peak order days and high-demand months
Highlighted top-selling pizza categories contributing to revenue
Detected underperforming pizzas that may need improvement or removal
Provided clear comparison between best and worst sellers

**Outcome**

This dashboard helps businesses:

Improve sales strategy
Optimize product offerings
Make informed business decisions
